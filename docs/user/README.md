# Nyx — User Guide

Nyx is a fast, native **SFTP / FTP / FTPS** client. Create connection profiles,
browse remote directories, and move files with drag-and-drop.

If you're a developer looking to build or contribute, see the
[project README](../../README.md) and [`docs/`](../README.md) instead — this
folder is for people *using* the app.

## Contents

1. **[Installing & first launch](install.md)** — download, get past macOS
   Gatekeeper, requirements.
2. **[Getting started](getting-started.md)** — create a connection, choose how to
   authenticate, browse and transfer files.
3. **[Troubleshooting & FAQ](troubleshooting.md)** — connection errors, host-key
   prompts, key passphrases, where your data is stored.

## Quick reference

| Action | Shortcut (macOS) |
|---|---|
| New connection | ⌘N |
| Toggle sidebar | ⌘B |
| Filter the current folder | ⌘F |
| Refresh listing | ⌘R |
| Close connection | ⌘W |
| Open / download selected | ↩ Enter |
| Go up a folder | ⌫ Backspace |
| Rename | F2 |
| Delete | Del |
| Select all | ⌘A |
| Copy path | ⌘C |
| Settings | ⌘, |
| Keyboard shortcuts | ⌘/ |

On Windows and Linux, use **Ctrl** wherever this guide says ⌘. Press **⌘/**
(Ctrl+/) any time inside the app for the full, always-up-to-date list.
